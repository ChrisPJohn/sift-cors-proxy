# Sift CORS Proxy

This is a lightweight, zero-cost Cloudflare Worker designed to securely proxy Git and RSS requests for the Sift Life OS.

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/ChrisPJohn/sift-cors-proxy)

## Deployment Instructions

1.  **Click the Deploy button above.**
2.  **Log in to or create a free Cloudflare account.**
3.  **Authorize Cloudflare to deploy the worker.**
4.  **Once deployed, copy your new Worker URL.** (It will look like `https://sift-cors-proxy.<your-username>.workers.dev`)
5.  **Paste that URL into the Sift Settings page.**

## Privacy Note

This proxy runs entirely on your personal Cloudflare account, ensuring absolute privacy for your Git tokens and Sift data.
